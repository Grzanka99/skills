# Skills

Reusable skills.

Each skill lives in its own directory and is defined by a `SKILL.md` file.

## Skills

- `fleet`: Resolves personal remote machines and provides task-driven SSH access using private inventory records.
- `opencode-subagent`: Delegates bounded tasks to explicitly requested OpenCode models.
- `planning`: Creates and maintains self-contained plan documents under `plans/<plan-name>/`.
